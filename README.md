## Getting Started

```bash
git clone https://github.com/x-tahosin/maxai-writer.git
cd maxai-writer
npm install
npm run dev
```

Create a `.env.local` file:
```
NEXT_PUBLIC_GEMINI_API_KEY=your_gemini_api_key
NEXT_PUBLIC_BNB_WALLET=your_bnb_wallet_address
```
